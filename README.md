# csharp-final-projects

Made by Salome Shiohvili for IT Academy C# course.

Projects
-----

- `Calculator`
	- Simple interactive calculator (add, subtract, multiply, divide) with input validation.

- `GuessNumber`
	- Number guessing game with difficulty levels and a CSV scoreboard (`scores.csv`).

- `Hangman`
	- Hangman word-guessing game with XML scoreboard (`hangman_scores.xml`).

- `ATM`
	- Console ATM app with user registration, login, deposit/withdraw, balance check and operation history.
	- Saves users and operation logs to JSON files (`users.json`, `logs.json`).


Notes
-----

- Each project is a standalone console app. Run them from their folder.
- Data files (JSON/CSV/XML) will be created next to the executable during first run if there is none.
